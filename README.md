## How to Run

1. Clone the project:

```bash
$ git clone git@github.com:helderburato/unsplash-scraper.git
```

2. Inside the root directory run the following code:

with `NPM`:
```bash
$ npm run scrape <username-example>
```

with `Yarn`:
```bash
$ yarn scrape <username-example>
```

_Note: Username arg. is required._
